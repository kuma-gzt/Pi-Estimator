# Pi Estimator
Yet another pi estimator by Monte Carlo methods
![Alt Text](./assets/pi-estimator.gif)

## Dependencies
gtk+-3.0
math.h C library

## Features
- Estimates the pi number using the dot-in-circle Monte Carlo method (the ratio of dots inside the circle to the total number of dots will approximately equal Ï€/4)
- Range of number of dots from 10k to 100k
- Option to select size of dots
